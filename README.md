# Cyber Arcade 🕹️👾

**Cyber Arcade** is an immersive, space-themed, gamified cybersecurity platform. It features a suite of interactive minigames designed to educate and entertain users while maintaining a sleek, premium glassmorphism UI.

## 🚀 Features

- **User Authentication:** Complete Login and Signup system with data persistence using LocalStorage. Includes persistent user profiles.
- **Daily Rewards & Streaks:** Engage users with daily login streaks and gift rewards.
- **Leaderboard:** Compete against others and track high scores globally!
- **Space-Themed Glassmorphism UI:** A sleek, modern user interface with dynamic hover effects, smooth transitions, and captivating micro-animations.
- **Responsive Design:** Playable across all screen sizes and devices.

## 🎮 The Games

Cyber Arcade features a diverse catalog of multi-level games, including:

- **🔐 Crypto Decoder:** Test your cryptography skills by decoding challenging ciphers.
- **🎣 Phishing Simulator:** Learn how to spot real vs. fake emails and links to protect yourself online.
- **🛡️ Vulnerability Scanner:** Identify and patch system vulnerabilities under pressure.
- **🧠 Memory Flips:** A classic memory card game with a cyberpunk twist.
- **🏹 Archery:** Sharpen your aim and timing with this target-practice game.
- **🎯 Tic Tac Toe:** The classic strategy game against an AI opponent.
- **🫧 Bubble Shooting:** Clear the board by matching colorful, cyber-themed bubbles.
- **🧩 Match the Design:** Test your visual memory and attention to detail.
- **💻 Bruteforce:** Learn the concepts behind password security by cracking simulated locks.
- **📝 Cyber Quiz:** Test your overall cybersecurity knowledge with multi-level quizzes!

## 🛠️ Technology Stack

This application is built entirely using the foundational web trio for a lightweight, speedy experience:
- **HTML5:** Semantic structure and layout.
- **CSS3:** Advanced styling, CSS Grid/Flexbox, Custom Properties, Glassmorphism, and Animations.
- **Vanilla JavaScript (ES6+):** Game logic, DOM manipulation, authentication state, and LocalStorage data management.

## 💻 How to Run Locally

If you want to run this application on your local machine:

1. Clone or download this repository.
2. Open your terminal and navigate to the project directory.
3. Start a local server. If you have Node.js installed, you can use:
   ```bash
   npx http-server .
   ```
4. Open the provided `localhost` URL in your web browser.

## 🌐 Hosting

To host this project for free using **GitHub Pages**:
1. Commit all your files and push them to a public GitHub repository.
2. Navigate to your repository's **Settings** > **Pages**.
3. Under the "Build and deployment" section, select the **main** branch and save.
4. Your site will be live at `https://USERNAME.github.io/REPOSITORY-NAME/` in a few minutes!
